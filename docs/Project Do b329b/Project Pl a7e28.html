1. Introduction

Scope

This project will aim to change the classic Monopoly game in a revolutionary way. The game has been based on a solid structure and has not changed in years. New board games offer players more complex game mechanics and with that, players engage in the game even more. Monopoly lacks this "change". New Monopoly will be an online web-based game. Players will be able to create or join a lobby to play the game. It will have a new game mode, where X number of players try to take down a single Monopoly player. Also, there will be new game mechanics such as inflation, foreign currencies, contracts, and more.

Non-Functional Issues

Since we will be using Unity Game Engine for developing our game, publishing the game on the web will be an easy task with Unity's WebGL support. Unity is a well-balanced game engine on behalf of performance. The game will run smoothly on the Web and will be playable with a mouse and keyboard.
